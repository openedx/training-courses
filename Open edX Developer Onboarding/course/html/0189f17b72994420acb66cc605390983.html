MFE Dev Tutorial

At the Open edX 2023 conference, Brian Smith & Adolfo Brandes (both from Axim) gave a tutorial on MFE development. There was no video from the tutorial because it consisted of following instructions and the instructors walking around the room helping out. We encourage you to give the tutorial a try!

The materials for this tutorial live in 2 repositories. The mfe-workshop-2023 repository contains instructions for setting up a development environment to work on Micro-Frontends. The frontend-app-workshop-example repository contains example applications.
