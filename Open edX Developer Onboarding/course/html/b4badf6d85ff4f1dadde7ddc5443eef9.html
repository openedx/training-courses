Books/Courses

      • [paid] JavaScript: The Good Parts is a solid text for learning good patterns in vanilla JS, but if I recall it does assume some programming knowledge already.
      • [paid] Learn JavaScript the Hard Way - similar to Learn Python the Hard Way mentioned in the previous section
      • [paid] Tyler McGuinness (ui.dev)
          • Highly rated courses on JS and React; recommended by a community member whose previous job paid for subscriptions to these courses
      • [free] Eloquent JavaScript
          • This is a book about JavaScript, programming, and the wonders of the digital.
      • [free] The JavaScript Way
          • Chapters 3, 2, 1... Code! through Write Functions specifically helped me get into JavaScript with 0 prior experience
      • [free] The Beginner's Guide to React
          • This course is for React newbies and anyone looking to build a solid foundation. It’s designed to teach you everything you need to start building web applications in React right away.
      • [free] React fundamentals
          • This course covers the features offered by React and explains their advantages and disadvantages relative to the other popular options available. It also explores important topics like React’s one-way reactive data flow, the virtual DOM, and JSX syntax for describing markup.
      • [free] React for Beginners
          • In this course, you will learn Modern JavaScript and React JS from absolute scratch. You will learn to make AJAX requests to get data from remote API and display it into your web application.
      • [free for 10 days] React 17: Getting Started
          • This course prepares the student to start developing web applications with the React JS library.
      • [free] React/Redux Masterclass
          • This course goes deeper into React and you’ll learn about things like:
              • Components
              • JSX
              • State
              • Events
              • Forms
          • You will also learn how to Refactor applications to be more testable and reusable (decoupled) and connect stateless views with stateless reducers through React containers.
