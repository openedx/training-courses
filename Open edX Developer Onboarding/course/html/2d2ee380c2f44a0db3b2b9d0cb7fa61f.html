Commit History

Now, let's say you want to look back at previous commits or compare changes. You can use the following command to see the commit history:

$ git log

The git log command shows a list of commits in reverse chronological order, allowing you to review the changes made over time. Their purpose is to ensure all commit messages are descriptive.
