Languages & Frameworks

We use a lot of different technologies - these technologies power our backend, frontend, and developer environments.

Backend

On the backend, we use a programming language called Python, and the Django framework.

      • Python is an easy-to-read functional programming language and one of the most popular languages for development (about 20% of all GitHub pull requests are in Python).
      • Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of web development, so you can focus on writing your app without needing to reinvent the wheel. It’s free and open source.

Frontend

On the frontend, we use a programming language called JavaScript, and the React framework.

      • JavaScript is a lightweight programming language, known for writing scripts on webpages. It can also be used in non-browser environments such as Node.js.
      • React is an open-source framework that allows you to build user interfaces out of individual pieces called components. Then, you can combine them into entire screens, pages, and apps.

Searching, Task Queue, and Code Mangement

  • The platform uses Elasticsearch for searching
  • Tasks queue asynchronously using Redis
  • We use Git and GitHub to manage our source code
