Once the DNS entries are set up, you can deploy and start your Open edX instance to Kubernetes by running:

tutor k8s launch

That’s it – once Kubernetes finishes setting up your resources, you have a running Open edX instance in a Kubernetes cluster.

See the Tutor k8s documentation to learn how to interact with and customize the resources in your instance.
