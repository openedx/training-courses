2. git remote

The git remote command is used to manage remote repositories that are associated with your local Git repository. It allows you to view, add, or remove remote repositories.

Here are some common git remote commands:

    • git remote -v : Displays a list of the remote repositories associated with your local repository, along with their URLs.
    • git remote add <name> <repository>: Adds a new remote repository to your local repository. You give it a name (e.g. upstream) to refer to the remote repository and provide the repository URL. To set the remote for upstream for the repo you cloned, use this command:
$ git remote add upstream https://github.com/openedx/edx-platform.git
