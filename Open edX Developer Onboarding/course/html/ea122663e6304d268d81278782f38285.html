Learning Git + Advanced Git

Here are some resources for learning Git that our community recommends:

      • Learn Git Branching 
      • The Git book
          • For beginners, chapters 1, 2, 3 and 5 are a must.
          • For advanced Git, chapter 7 is great - reset, stashing, searching commit (and diff!) history, and checkout are all important.
          • Reflog is a crazy concept but worth learning if you want to become a Git expert.
          • Similarly, bisecting! Bisecting is sometimes comes in clutch when you’re trying to find a commit that breaks something.
      • Git from the Bottom Up - This is a more advanced resource.
