All Operating Systems

After installing the packages using Debian Commands or OSX Commands, you need to edit the PATH variable in your shell configuration file in order to have make and pip work properly, as well as to have python3 properly alias to python:

# in .bashrc, .zshrc, or other shell configuration file
if command -v pyenv 1>/dev/null 2>&1; then
eval "$(pyenv init -)"
fi
if type brew &>/dev/null; then
FPATH=$(brew --prefix)/share/zsh-completions:$FPATH
fi
export PATH="$(pyenv root)/shims:$PATH"
export PATH="$(pyenv root)/bin:$PATH"
export PATH="/opt/homebrew/opt/make/libexec/gnubin:$PATH"

You can apply these changes by running:

$ source ~/.zshrc

or whatever the path to your shell configuration file is. Running `make --version` should give you a version that’s 4.4 or above, and if you’re on an Apple Silicone device, will have a line that starts with “Built for aarch64-apple-darwin*”.

$ which python

Should give you a path that has your pyenv on it (for example, /Users/username/.pyenv/shims/python)
