Pre-requisites

You will need access to a supported server or virtual machine with enough RAM, compute power, and disk space for your specific needs. The VM should be configured so that it is accessible from the internet. See the Tutor requirements page for more details.

In our experience, most Open edX hosting providers choose to run their Open edX instances on a cloud server. Most cloud servers have multiple layers of security monitored by dedicated development teams and provide more reliable and consistent access as they are not affected by local power interruptions or catastrophic hardware failures. Popular choices for cloud providers include AWS, Azure, Google Cloud Platform, and DigitalOcean.

You will also need access to the DNS settings of the domain that you will use for your instance.
