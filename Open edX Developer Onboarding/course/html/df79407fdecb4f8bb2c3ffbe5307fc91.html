To Learn More:

When developing using Tutor, you’ll encounter many plugins. It’s a good idea to inspect the documentation of these plugins to know what you’re using! One very useful plugin is the tutor-mfe plugin. There are many other plugins available in the Overhang.io GitHub organization.

Interested in writing your own plugin? We hope you are! Check out cookiecutter-tutor-plugin - it’s a great jumping off point for your new plugin.
