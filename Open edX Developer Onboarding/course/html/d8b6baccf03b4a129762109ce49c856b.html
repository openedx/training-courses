Should I get involved?

If you are passionate about developer environments, making the dev onboarding experience smoother for everyone, or interested in learning something new… yes! Join the Tutor Users Group!

See the Tutor Users Group wiki page to learn more about how you can use Slack to get in touch with the group. You can find out when they meet by viewing the working group calendar (hint: use the (+) button in the bottom right to add this calendar to yours!).
