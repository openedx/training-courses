Welcome to Module 2: Developer Setup

In this module, you’ll learn about our main development environment. By the end of this module, you will have a working local environment setup with one or both of the environments.

What’s a developer environment?

From Umbraco: A development environment in software and web development is a workspace for developers to make changes without breaking anything in a live environment… by using a development environment for your website, you'll have a place to make all of the changes you want, without the users of your website noticing.

We present two development environments: Tutor and Devstack. Tutor is the recommended development environment for backend development and for running and operating a site (more on this in Module 3). Devstack is used by people developing Micro Front-ends (MFEs).
