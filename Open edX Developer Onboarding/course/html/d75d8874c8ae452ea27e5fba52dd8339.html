Module 6 Table of Contents

In this module we’ll explore some talks, tutorials, and references that will help you learn more about the tools we use in the Open edX project. We’ve split these into three categories:

  1. General Resources - tools that are standard across the software development industry - things like Git (version control software), Python/Javascript (programming languages), and Django/React (code frameworks).
        • Learning Git + Advanced Git
        • Opening A Pull Request from a Fork
        • Python courses & tutorials
        • Django, JavaScript, React Courses & tutorials
        • Make Your Environment Work For You

2. Open edX-specific Resources - tutorials and talks, frequently recorded at Open edX Conferences, that dive into topics relevant to all developers that are specific to our project. These resources are varied and cover things such as how to write documentation for our software stack, how to properly internationalize Open edX code, and how to write your own micro-frontend.

      • Intro to Open edX Platform & Community
      • Tutor
      • Architecture & XBlocks
      • MFE development
      • Internationalization (i18n)
      • Accessibility (a11y)
      • Documentation

3. Advanced Resources - also Open edX-specific, but cover topics you may not be diving into right at the beginning of your Open edX development journey. These topics, such as how to use Hooks & Extensions, may be worth coming back to as you become a more seasoned Open edX developer.

      • Mobile dev tutorial (2023)
      • Hook & Extensions
      • Conference Talks
