Frontend Architecture

Let’s learn more about the Frontend Architecture in this video by David Joy (formerly Principal Software Engineer at 2U/edX).

Please Note: While the video was recorded in 2019, it is still very relevant, with a few caveats. The technologies named in OEP-11 will - and already have - changed.  For instance, Enzyme. Otherwise, it's all pretty much still applicable.  What's changing is "just" how many single-page apps (SPAs) we'll have. Previously, each MFE has been its own SPA; going forward there'll be a small number of SPAs that in turn host them.
