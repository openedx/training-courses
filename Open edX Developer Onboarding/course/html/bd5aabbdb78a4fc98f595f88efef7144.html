Django Settings

As mentioned above, a large number of settings are implemented as Django settings. The edx-platform documentation contains a list of available Django settings. Note, however, that the list does not contain Django settings that act as toggles (we’ll discuss toggles in the next section) and only contains edx-platform settings. Settings for other services and apps are not listed in this document; for those, you’ll need to inspect the settings files in the GitHub repository corresponding to the service/app you’re looking for.

To modify custom Django settings in Tutor, you need to create a Tutor plugin. The Tutor documentation contains a tutorial about it, so we are not going to go through all the steps here.

Once your plugin with custom settings is ready and installed, you need to save the Tutor configuration with Tutor config save and restart it by running tutor local restart for local installation, or tutor k8s start when deployed in Kubernetes.
