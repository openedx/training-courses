Welcome to the Open edX Project's Developer Onboarding Course!

This course is for developers who already have experience writing code and will learn how to apply that knowledge to the Open edX project. Also, site operators with a familiarity of the command line will learn how to deploy and run their own Open edX instance.

This course assumes you have familiarity with some of the basics of coding - knowledge of at least one programming language will be very helpful as you go through this course. We provide links to some recommended resources for getting up to speed with our tech stack (Python/Django, JavaScript/React) in the last module of this course.

After taking this course, you will be able to:

        • Identify the technical architecture of the Open edX platform
        • Create and run a development environment
        • Determine the steps to deploy and run your own Open edX instance
        • Recognize how we do development on the Open edX project
        • Demonstrate how to contribute your code back to the main project.

If you'd like more context on the Open edX project - including project history, project context, and an overview of the different roles for Core Contributors (not just via code!) - we recommend you start with Intro to the Open edX Project & Contributing. This course assumes that you know the content covered in the Intro to the Open edX Project course.
