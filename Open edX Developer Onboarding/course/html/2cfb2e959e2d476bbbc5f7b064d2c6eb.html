OSX Commands

Next, install Homebrew via the command line and Docker for Mac via their online interface.

# OSX: install Homebrew
/bin/bash -c "$(curl -fsSL
https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
# Install packages
brew install make
brew install python
brew install libyaml
brew install pyenv
pyenv install 3.11.10 # or current Open edX Python version
