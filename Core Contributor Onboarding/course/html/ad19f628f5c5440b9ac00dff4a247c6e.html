We'd like to take this opportunity to celebrate some of the many, many wins that have come out of the Core Contributor program.

These include:

  • Proposal and implementation of the Hooks Extension Framework (Felipe Montoya, Maria Grimaldi)
  • Building a formalized release testing process and conducting Quality Assurance (QA) work for the Open edX named releases, starting with Nutmeg (Dean Mathew, Pierre Maihot, and others)
  • Implementing LTI functionality and getting the platform certified as LTI Advantage Complete (Giovanni Cimolin da Silva)
  • Formalizing the Core Contributor program and enabling CCs to talk to each other and share their status in a systematic way (Dean Mathew, Xavier Antoviaque)
  • Removing "XModules", a legacy code module and one of the biggest (and most complicated!) hurdles towards advancing our platform (Usman Khalid)
  • Tons of work by the translations teams to add machine translations and review them to get key languages at 90% or more (see the Core Contributors page for a list of all translators and their target language(s))
  • And so much more!
