The following video is a presentation given by Feanil Patel (Axim Software Architect) that will help you understand the overall technology stack of the Open edX platform.

Here are some quick notes about what you'll find in this video. Reading it before checking the architecture video would be helpful so you can follow along with the video better.

Tech stack

  • The Open edX platform is mainly built with Django/Python REST for backend services, react micro-frontends, and legacy server-rendered templates for frontend.
  • MySQL / MongoDB for persistent storage.
  • Elasticsearch for searching.
  • Memcached for caching.
  • Redis for asynchronous jobs queue.

Platform layout

  • While the platform contains many services such as MFEs and libraries, the core of the platform lives in the edx-platform and frontend-platform repositories.
  • MFEs live in different repositories across the openedx organization, while some libraries live within the edx-platform repository.

Extensions

  • Besides the platform layout previously mentioned, there are a number of other services which enhance the platform core. Eg., ecommerce or discovery services. Those projects are often found under the openedx organization.

For a more detailed explanation, please refer to the Open edX Documentation: Platform overview. This section serves as an introduction to the project.
