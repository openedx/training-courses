This course is mainly focused on the history of the project, and how to contribute. With that said - ANYONE should take this course, especially you!

Q:Do I need to know how to code to contribute to the Open edX platform?

A:Not at all! The project needs you! Whatever your skill set - for example: UX designer, marketing specialist, project manager, writer (technical documentation, blog posts, etc), and many more - there is a place for you within the Open edX project.
