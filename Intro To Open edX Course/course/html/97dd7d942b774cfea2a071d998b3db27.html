How can I contribute to the Product Roadmap?

There are two main ways to contribute to the Roadmap.

  1. If you are working on a Feature that youd like to be considered for inclusion in the Core Product, fill out an Issue Submission ticket.
  2. If youd like to request a new feature that isnt already part of the project, fill out a Feature Request ticket. Please note - submitting a ticket doesnt guarantee that the feature will be built, but it will be added to the backlog for consideration by the Product Working Group.
