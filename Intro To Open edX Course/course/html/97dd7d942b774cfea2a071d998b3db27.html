How can I contribute to the Product Roadmap?

There are two main ways to contribute to the Roadmap.

  1. If you are working on a Feature that you’d like to be considered for inclusion in the Core Product, fill out an Issue Submission ticket.
  2. If you’d like to request a new feature that isn’t already part of the project, fill out a Feature Request ticket. Please note - submitting a ticket doesn’t guarantee that the feature will be built, but it will be added to the backlog for consideration by the Product Working Group.
