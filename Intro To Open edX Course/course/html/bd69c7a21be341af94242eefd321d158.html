Small bugs/fixes: Currently, the project does not maintain a list of small bugs or fixes. However, you can check out GitHub Issues on each repo to see if there are work items you can pick up.

Reporting bugs: You can report bugs using GitHub Issues in the relevant repository. If you're not sure how to start, post in the forums for advice.

New feature proposals: We have a process for proposing a new feature, which begins with adding a ticket to the Platform Roadmap issue tracker. We're currently in the process of writing up more detailed instructions; feel free to post in the forums if you need more guidance until we've had those written up.

Working Groups: Working Groups are the best place to find things to work on; as of May 2023 there are numerous working groups pertaining to code, including:

Developer Experience
Frontend
Data
Paragon
DevOps
Large Instances
Code Deprecation (DEPR)
Security

See the Working Groups section for more detail. If you can't make a synchronous Working Group meeting, post in their Slack room for information on where they could use help.
