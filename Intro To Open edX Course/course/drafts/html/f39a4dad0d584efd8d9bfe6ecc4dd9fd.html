FAQ

How do you find the maintainer of a repo?

Not all repos currently have named maintainers, but for the ones that do, you can find maintainers via the Open edX Backstage tool. In this tool, you can see details about a repo:

Here, you can see the repo "Blockstore" is maintained by a group called "blockstore-maintainers".

When should you reach out to a maintainer?

Common reasons to reach out to a maintainer include: I found a bug!; How can I implement X using this library?; I have a suggestion for a new feature; I'd like to build a new feature and am looking for a place to start; I'm building a new feature and need some architecture review.

How you reach out to a maintainer?

You can open a GitHub issue in the appropriate repo and tag the maintainer team that is detailed in Backstage. You can post on the forums, as well.
